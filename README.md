🛠️ 代码质量与规范 (ESLint + Prettier)

本项目集成了 ESLint 与 Prettier，旨在为 Vue3 + Vite + TypeScript 项目提供严格的代码质量约束和统一的代码风格。
主要作用：
ESLint：负责代码质量检查，规避潜在逻辑错误（如未使用的变量、类型错误等）。
Prettier：负责代码格式化，强制执行统一的代码风格（如引号、缩进、分号等），消除团队协作中的风格差异。
常用命令：
pnpm run lint：检查并自动修复代码中的格式问题与部分逻辑错误。
